## How Each Lesson Works

Every lesson page gives you:

1. **🎥 Video** — watch the lesson on YouTube (embedded at the top)
2. **📖 Chapter** — the full lesson notes: concepts, steps, tables, analogies
3. **✅ Quiz & Assignment** — test yourself and practice hands-on
4. **📦 Resources** — slides, workflow files, and GitHub links
